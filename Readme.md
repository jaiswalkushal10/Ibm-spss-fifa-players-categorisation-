Project Overview:
Here we have used a dataset from Kaggle.
Our dataset contains various football icons and players around various competitions and countries and their stats like physical features and form.
the dataset originally contained variety of fields, Which we filtered.
Our project uses some fields to create some fields like BMI , Age class , Potential Class, BMI class.

Project Objective;
All the fields we have created will help a coach to understand current value of players and help them to know whether to sign one or not.
We have used multiple derive nodes to perform our program.
